# 🖼️ Interactive Image Gallery

A modern and responsive image gallery with advanced features like image preview, navigation controls, and automatic slideshow.

## 🚀 Features
- Responsive grid layout
- Hover animations
- Click to preview images in full screen
- Next/Previous navigation controls
- Automatic slideshow functionality
- Clean and professional UI design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
